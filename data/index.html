ESP32 Captive Portal

Feature-Rich Wi-Fi Configuration Interface

Portal is running

?? Wi-Fi Features

  • Access Point with captive portal
  • Open network (no password) or WPA2 secured
  • DNS hijacking for captive portal detection
  • Customizable SSID and channel
  • Multiple device connections (up to 4)

?? Web Server Features

  • Static file serving from SPIFFS
  • Automatic MIME type detection
  • Wildcard URI handlers
  • HTTP methods: GET, POST, PUT, DELETE
  • Custom API endpoint support

?? Device Compatibility

  • Android (generate_204 detection)
  • iOS/Mac (hotspot-detect.html)
  • Windows (ncsi.txt, connecttest.txt)
  • Automatic redirects to portal
  • Proper captive portal responses
